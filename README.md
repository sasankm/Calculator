# Calculator

## Installation steps

-clone or download the zip 
-open calculator in your IDE
-open cmd or terimanl in that particular IDE
- In 1st termianl or cmd type: 
        1.cd calculator-client
        2.npm install
        3.npm start
- In 2nd termianl or cmd type:
        1.cd calculator-server
        2.npm install
        3.npm start
